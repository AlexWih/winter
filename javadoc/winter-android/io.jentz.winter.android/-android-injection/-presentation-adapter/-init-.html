PresentationAdapter()

Adapter that operates on the GraphRegistry and requires a root component with a "presentation" named subcomponent that has a "activity" named subcomponent. The adapter adds the activity instance to the activity dependency graph.

The activity graph gets disposed on AndroidInjection.onActivityDestroy but the presentation dependency graph gets only disposed when Activity.isFinishing returns true so the presentation dependency graph survives configuration changes.
