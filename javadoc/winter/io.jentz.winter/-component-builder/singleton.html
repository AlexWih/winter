singleton

inline fun <reified T : Any> singleton(qualifier: Any? = null, generics: Boolean = false, override: Boolean = false, noinline block: ProviderBlock<T>): Unit

Register a singleton scoped provider for an instance of type T.

This is syntactic sugar for provider with parameter scope = singleton.

Parameters

qualifier - An optional qualifier.

generics - If true this will preserve generic information of T.

override - If true this will override a existing provider of this type.
